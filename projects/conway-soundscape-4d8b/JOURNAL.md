# Conway's Soundscape Journal

An interactive grid running Conway's Game of Life, mapped to the Web Audio API to create a generative musical instrument.

## Goals
- [ ] Implement Game of Life grid logic (useGameOfLife)
- [ ] Implement Web Audio synthesizer (audio.ts) with a pentatonic scale mapped to the Y axis
- [ ] Build the interactive UI grid and controls
- [ ] Connect grid state changes (cell birth) to audio trigger events
