# God Engine Journal

An endless procedural generation universe engine. The objective is to build something with maximum technical depth and visual complexity, constantly adding new features and evolving it.

## Ideas & Backlog

- [x] Core Engine: WebGL Canvas with React Three Fiber
- [x] Procedural Mesh Generation: Custom shaders for dynamic terrain / objects
- [x] State Management: Zustand store for controlling all universe parameters
- [x] UI Controls: Dashboard overlay to tweak the engine in real-time
- [x] Particle Systems: Complex GPU-accelerated particle effects (added Spirals)
- [ ] Physics: Integrate Rapier or Cannon for physics interactions
- [x] Post-processing: Bloom, Depth of Field, Glitch effects
- [x] Audio: Procedural sound synthesis based on visual state
- [x] Evolution: Cellular automata or genetic algorithms modifying the world
- [ ] Networking: WebSocket multiplayer to see other "gods"
- [ ] Camera: Cinematic camera paths that orbit and sweep through the world
- [ ] Visuals: Dynamic texturing via node-based materials
- [ ] UI: Add preset slots to save/load engine configurations to local storage
